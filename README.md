# ChatProgram
Chatting Program on Ubuntu Linux &amp; QT Creator

# How to running?



// 1st

 1. set on terminal
 2. $ cd ChatProgram
 3. $ ./server ()
 4. new terminal
 5. $ cd build-client-Destop-Debug
 6. $ qmake
 6. $ make
 7. $ ./client

// 2nd
 1. Install Qt creator
 2. $ ./server
 3. Excute the Qt Creator And Open the client.pro
 4. Run ui
